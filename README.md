Это frontend-приложение на базе Next.js 16.1.1!

## Стек:

framework: Next.js
language: TypeScript
styling: Tailwind CSS
data: локальный data.json

# Структура:

app/ - основная страница
components/ - UI-компоненты
components/icons/ - inline-SVG компоненты
public/ - иконки, изображения
types/ - файлы для типов
store/ - хранилище управления состоянием
data.json - данные проекта (случайные текста)

# Особенности:

адаптивный дизайн
полная TS типизация
вся логика в Zustand хранилище
статистика скорости и точности печати

проект - решение задачи с заготовленным дизайном
дизайн: https://www.frontendmentor.io/challenges/typing-speed-test
